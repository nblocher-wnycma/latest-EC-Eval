Farmstead Evaluation Generator - v26 Modern UI

How to run:
1. Right-click the ZIP file and choose Extract All.
2. Open the extracted folder.
3. Double-click Run_Farmstead_Full_Narrative_Generator.bat.

This version keeps the working narrative/document generation code from v26 and only updates the program interface.

Visual updates:
- Cleaner header and toolbar.
- More current colors and spacing.
- Better button labels.
- Cleaner tabs and section cards.
- More readable text boxes and input fields.

Workflow:
1. Fill out Basic Info.
2. Use the Quick Build Wizard.
3. Click Build ALL Sections from Quick Wizard.
4. Review the section narratives.
5. Generate the Word document.
